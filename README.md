# p5_pbd_sim_playground
Playground for 2d pbd and some physics sim in p5.js


### Constraint :
- Distance
- Collision : sphere, plane
- Bending : line

### External Force/Factor :
- Gravity
- Friction : static, kinetic
- Drag
- Buoyancy
